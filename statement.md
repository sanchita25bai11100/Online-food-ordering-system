##  Problem Statement
Users need a simple and efficient way to order food through a console-based system that displays menus, collects details, and generates an automated bill.

---

##  Scope of the Project
This project focuses on:
- Displaying categorized menus
- Allowing users to select items
- Handling multiple orders
- Calculating totals
- Displaying an order summary

---

##  Target Users
- Students
- Beginners learning Python
- Small snack centers or canteen-style setups
- Anyone learning console-based system design

---

##  High-Level Features
- Veg & Non-Veg module selection  
- Four-category menu design  
- Price mapping using dictionaries  
- List-based order storage  
- Bill calculation  
- Summary generation  
- Clear user navigation
