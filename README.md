## Project Overview
This is a console-based Online Food Ordering System that allows users to enter their details, choose Veg or Non-Veg menus, select items from four categories (Snacks, Meals, Beverages, Desserts), place multiple orders, and view a final bill summary.  
It is designed to demonstrate basic programming concepts and object-oriented design.

---

##  Features
- User detail collection (Name, Address, Mobile Number)
- Veg & Non-Veg menu modules
- Four food categories in each module
- Prices displayed clearly
- Multiple item ordering
- Real-time bill calculation
- Final order summary with user details
